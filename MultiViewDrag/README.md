Qt 拖放操作演示项目

项目概述

这是一个基于 Qt 框架开发的拖放操作演示应用，展示了 Qt 中不同视图控件的拖放功能配置与实现。该项目不仅是一个学习 Qt 拖放机制的实用示例，也是理解 Qt 事件过滤器、视图控件操作的重要教学资源。

核心功能

1. 多视图控件支持
2. 列表视图 (`QListWidget`)：支持项目的拖放操作
3. 树状视图 (`QTreeWidget`)：支持层级结构的拖放操作
4.表格视图 (`QTableWidget`)：支持单元格的拖放操作

拖放功能配置
控制拖放时是否覆盖目标项目
启用或禁用拖放功能
配置不同的拖放行为模式
设置默认的拖放动作（复制、移动、链接等）

增强交互体验
通过 Delete 键快速删除选中项目
支持将本地图片文件拖放到应用中预览
修改配置后立即应用到选中的视图控件

关键技术
使用 `eventFilter` 实现对不同控件事件的统一处理
通过 `QAbstractItemView` 抽象类统一处理不同类型的视图控件
实现了树状视图的递归删除功能
